## 如何使用issue進行討論
請參考以下教學
- [Mastering Issues](https://guides.github.com/features/issues/)
- [Mastering Markdown](https://guides.github.com/features/mastering-markdown/)
- Hint: 可以直接把整個R chunk貼上來, 也可以貼圖
